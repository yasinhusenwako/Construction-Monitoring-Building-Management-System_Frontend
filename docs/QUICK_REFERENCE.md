# Status Workflow - Quick Reference

## 🎯 Quick Actions

### Add Status
```
1. Click "+ Add Status"
2. Type name
3. Press Enter
```

### Edit Status
```
1. Hover over status
2. Click ✏️ (edit icon)
3. Change name
4. Press Enter
```

### Delete Status
```
1. Hover over status
2. Click 🗑️ (delete icon)
3. Confirm
```

### Reorder Status
```
1. Click ⬆️ to move up
2. Click ⬇️ to move down
```

### Save Changes
```
1. Click "Save Changes"
2. See success message
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Enter | Save edit/add |
| Escape | Cancel edit/add |

## 🎨 Visual Indicators

| Color | Meaning |
|-------|---------|
| 🔵 Blue | Edit action |
| 🔴 Red | Delete action |
| 🟢 Green | Save action |
| ⚪ Gray | Cancel action |

## 📋 Three Streams

1. **Stream A: Projects**
2. **Stream B: Space Allocation**
3. **Stream C: Maintenance**

Each stream has independent workflow statuses.

## 💡 Tips

- Statuses are numbered automatically (1, 2, 3...)
- Hover to reveal action buttons
- Changes save to localStorage
- Refresh page to verify persistence
- Works in light and dark mode
